# VentureAI

## Project Overview
VentureAI is an AI-based startup networking and investment platform that connects entrepreneurs and investors.

## Features
- Startup registration and management
- Investor connections
- AI-based startup recommendations
- Admin dashboard

## Technologies Used
- Python
- FastAPI
- HTML
- CSS
- JavaScript
- MySQL

## How to Run
1. Install required packages
2. Configure the database
3. Run the application
