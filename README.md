# VentureAI

## Project Overview
VentureAI is an AI-based startup networking and investment platform that connects entrepreneurs and investors.

## Features
- Startup registration and management
- Investor connections
- AI-based startup recommendations
- Admin dashboard

## Technologies Used
- Python
- FastAPI
- HTML
- CSS
- JavaScript
- MySQL

## How to Run
1. Install the required packages
2. Configure the MySQL database
3. Run the FastAPI application
4. Open the application in your browser
